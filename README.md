# BigQuery GA4 E-commerce Analytics

This repository includes BigQuery SQL projects focused on GA4 e-commerce data analysis.

The projects cover event-level data preparation, funnel analysis, landing page conversion analysis, and engagement-purchase correlation using the public GA4 sample e-commerce dataset.

## Tools Used

- Google BigQuery
- SQL
- GA4 Public Dataset
- Looker Studio

## Dataset

`bigquery-public-data.ga4_obfuscated_sample_ecommerce.events_*`

## Projects

### 1. GA4 Flat Events Table

Prepared a clean event-level table from GA4 raw event data for reporting and dashboarding.

### 2. Funnel by Day and Channel

Analyzed e-commerce funnel performance by date, source, medium, and campaign.

### 3. Landing Page Conversion Analysis

Measured landing page performance based on sessions, purchases, and purchase conversion rate.

### 4. Engagement and Purchase Correlation

Analyzed the relationship between session engagement, engagement time, and purchase behavior.
